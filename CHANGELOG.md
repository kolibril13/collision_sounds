# Changelog

- **Audio Markers collection**: 
  - Hidden in rendered view (`hide_render = True`). 
  - Added "Show Markers in Viewport" toggle in the addon UI to control viewport visibility.
  - Reduced marker sphere sizes: slow collisions 0.08, fast 0.40 (was 0.16 / 0.50).
  - More vibrant marker colors: HSV saturation 2.0, emission strength 2.0 (hue unchanged).
